Welcome to ManyCode

ManyCode is a non-profitable software organization dedicated to helping people using open-source softwares. We work on small as well as on large projects designed for general purpose. Most of our projects are written in Python but we also work with other languages like C, C++ and Java. Our projects are hosted on GitHub in our software repository.
